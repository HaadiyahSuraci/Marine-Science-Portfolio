# Marine Science Portfolio

Independent marine science portfolio focused on benthic ecology, marine biodiversity, habitat visualization, and phytoplankton DNA barcoding.

Research interests include substrate-dependent species distribution, ecological interactions within benthic ecosystems, and the development of interactive marine habitat simulations for scientific and educational applications.

This repository contains research notes, ecosystem studies, visualization projects, and computational marine science experiments.
